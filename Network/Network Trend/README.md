# 1.7 Network Trends (네트워크 트렌드)

---

## 학습 목표

- 조직과 소비자에게 영향을 미치는 **최신 네트워크 트렌드** 4가지를 설명할 수 있다
- **BYOD**의 개념과 의미를 설명할 수 있다
- **Online Collaboration** 도구의 역할을 설명할 수 있다
- **Cloud Computing**의 4가지 유형을 구분할 수 있다
- **Smart Home**, **Powerline Networking**, **Wireless Broadband**의 특징을 설명할 수 있다

---

## 1. Recent Trends (최근 트렌드)

> 네트워크의 역할은 새로운 기술과 End User 장치들이 시장에 끊임없이 출시됨에 따라 **지속적으로 변화하고 적응**해야 한다.

조직과 소비자에게 영향을 미치는 **4가지 새로운 네트워크 트렌드**:

| 트렌드 | 설명 |
|--------|------|
| **BYOD** (Bring Your Own Device) | 개인 기기를 업무/학습에 사용 |
| **Online Collaboration** | 네트워크를 통한 협업 |
| **Video Communications** | 화상 통신 |
| **Cloud Computing** | 클라우드 기반 서비스 |

---

## 2. BYOD (Bring Your Own Device)

> 사용자가 **자신의 개인 기기를 사용**하여 더 많은 기회와 높은 유연성을 가질 수 있게 해준다.

BYOD는 최종 사용자가 자신의 개인 도구로 정보에 접근하고 소통할 자유를 제공한다.

### BYOD에서 사용 가능한 기기

- **Laptops** (노트북)
- **Netbooks** (넷북)
- **Tablets** (태블릿)
- **Smartphones** (스마트폰)
- **E-readers** (전자책 단말기)

>  **BYOD = 어떤 기기든, 어떤 소유권이든, 어디서든 사용 가능**
> ("any device, with any ownership, used anywhere")

---

## 3. Online Collaboration (온라인 협업)

> 네트워크를 통해 **공동 프로젝트에서 다른 사람들과 협업하고 함께 작업**하는 것

- 협업 도구(예: **Cisco WebEx**)는 사용자가 즉시 연결하고 상호작용할 수 있게 해줌
- 협업은 **비즈니스와 교육 분야**에서 매우 높은 우선순위를 가짐

### Cisco Webex Teams 기능

- 인스턴트 메시지 전송 (send instant messages)
- 이미지 공유 (post images)
- 영상 및 링크 공유 (post videos and links)

---

## 4. Cloud Computing (클라우드 컴퓨팅)

> 인터넷 상의 서버에 **개인 파일을 저장하거나 데이터를 백업**할 수 있게 해주는 기술

- 애플리케이션도 클라우드를 통해 접근 가능
- 기업이 **전 세계 어디서든 어떤 기기로도** 서비스를 제공할 수 있게 해줌
- 클라우드 컴퓨팅은 **데이터 센터** 덕분에 가능
  - 자체 데이터 센터를 구축할 여유가 없는 소규모 기업 → 대형 데이터 센터 기업으로부터 **서버와 스토리지 서비스를 임대(빌려서)** 사용

### 클라우드의 4가지 유형

| 유형 | 설명 | 특징 |
|------|------|------|
| **Public Cloud** | 일반 대중에게 **종량제 또는 무료**로 제공 | 구글 드라이브, iCloud |
| **Private Cloud** | **특정 조직 전용** (정부, 기업 등) | 정부 전용 클라우드 |
| **Hybrid Cloud** | **Public + Private 혼합** 구성 | 각 부분은 독립적이지만 동일한 아키텍처로 연결 |
| **Custom Cloud** | **특정 산업 맞춤형** 구축 | 의료(healthcare), 미디어(media) 전용 |

```
Public  → 누구나 (종량제/무료)
Private → 특정 조직만 (정부 등)
Hybrid  → Public + Private 혼합 (각 부분 독립, 동일 아키텍처 연결)
Custom  → 특정 산업 맞춤 (의료, 미디어 등) → private 또는 public 가능
```

## 5. Technology Trends in the Home (가정 내 기술 트렌드)

> **스마트 홈 기술** = 일상적인 가전제품에 기술을 통합시켜 다른 기기들과 서로 연결되게 하는 성장하는 트렌드 → **IoT (사물인터넷)**

### 스마트 홈 예시

> 오븐이 **캘린더와 통신**하여 내가 집에 돌아올 시간에 맞춰 자동으로 요리를 시작
> → 오븐이 캘린더와 통화해서 집에 돌아오는 시간에 맞게 가동 시작

- 현재 집 안의 **모든 방**에 적용될 수 있도록 개발 중

### 그림 설명

```
[집] ─── Cloud ─── [스마트폰]
 |                      |
[자동차] ←────────────────┘

집, 자동차, 스마트폰이 클라우드를 통해 연결 = IoT
```

## 6. Powerline Networking (파워라인 네트워킹)

> **기존 전기 콘센트를 이용해 데이터를 전달하는 네트워킹 방식**

- 데이터 케이블 설치가 어렵거나 **Wi-Fi 신호가 닿지 않는 곳**에서 특히 유용
- **표준 파워라인 어댑터**를 콘센트에 꽂기만 하면 LAN에 연결 가능
- 특정 주파수로 데이터를 전송

### 언제 쓰나?

```
Wi-Fi가 안 닿는 구석방
→ 콘센트에 어댑터 꽂으면 바로 LAN 연결!
→ 새 케이블 공사 필요 없음 (전기선이 곧 네트워크 선)
```

### 케이블 색깔 의미 (그림)

- **주황선**: Powerline Connection (전기선 활용)
- **보라선**: Wired Connection (일반 유선)

---

## 7. Wireless Broadband (무선 브로드밴드)

> DSL과 케이블에 이어, **가정과 소규모 사무실을 인터넷에 연결하는 또 다른 무선 옵션**

| 특징 | 내용 |
|------|------|
| **주요 환경** | 농촌·오지 등 유선 인프라가 부족한 곳 |
| **WISP** | Wireless ISP → 가입자를 **지정된(지정됨) 액세스 포인트나 핫스팟**에 연결 |
| **기술** | 스마트폰과 동일한 **셀룰러 기술** 사용 |
| **설치** | 집 **밖에 안테나 설치** → 집 안 기기들에 유선·무선으로 연결 제공 |

### 구조

```
[기지국/안테나 타워]
        |
   (무선 신호)
        |
[집 외부 안테나 설치]
        ↓
[집 안 기기들에 유선·무선으로 연결 제공]
```

## 배운 점

- **BYOD**는 단순히 편의성의 문제가 아니라, 네트워크가 다양한 기기를 수용할 수 있는 **Scalability**와 **Security** 문제와도 직결된다
- **Cloud Computing**은 소규모 기업도 대형 인프라를 빌려 쓸 수 있게 해 진입 장벽을 낮췄으며, 4가지 유형의 차이를 명확히 알아야 한다
- **Smart Home = IoT**로 이해하면 쉽다. 가전제품이 클라우드를 통해 서로 연결되고 자동화되는 것이 핵심이다
- **Powerline Networking**은 Wi-Fi도 케이블도 안 되는 환경에서의 현실적인 해결책이다
- 네트워크 트렌드는 기술의 발전에 따라 계속 변화하므로, 네트워크 인프라도 유연하게 적응해야 한다
