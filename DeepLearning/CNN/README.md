## 1. CNN은 어떻게 인간의 사고를 흉내냈을까?

CNN의 핵심 아이디어는 **국소적 연결(Local Connectivity)** 이다.

- FC(Fully Connected) 레이어는 모든 입력 픽셀과 모든 뉴런이 연결되어 있어 연산량이 방대하고, 이미지의 공간적 구조를 무시한다.
- Convolution 레이어는 인접한 픽셀들(국소 영역)만을 보고 특징을 추출한다.
- 인간의 시각 피질도 시야의 일부 영역만 담당하는 뉴런들이 계층적으로 구성되어 있으며, CNN은 이를 모방한다.

---

## 2. 이미지 인식에서 FC 레이어가 가지는 문제

FC 레이어를 이미지에 그대로 적용하면 여러 문제가 발생한다.

- **과도한 세밀함**: 개별 픽셀 단위로 분석하므로 이미지의 전체적인 구조나 패턴을 파악하기 어렵다.
- **위치 정보 손실**: 이미지를 1차원으로 펼치면(flatten) 픽셀 간의 위치 및 배치 정보가 사라진다.
- **파라미터 폭발**: 고해상도 이미지일수록 파라미터 수가 기하급수적으로 증가한다.

---

## 3. Convolution의 동작 방식

Convolution 연산은 작은 크기의 **필터(Filter/Kernel)** 를 이미지 위에 슬라이딩하며 특징을 추출한다.

- 필터 크기: 일반적으로 3×3
- **가중치(weights)와 바이어스를 재사용**하여 파라미터 수를 크게 줄인다 (파라미터 공유)
- 연산 결과로 **특징 맵(Feature Map)** 이 생성된다
- 필터의 종류에 따라 수직 엣지, 수평 엣지 등 서로 다른 특징을 추출할 수 있다

```
입력 이미지 * 필터 → 특징 맵 (Feature Map)
```

---

## 4. 다채널 입력에 대한 Convolution

실제 이미지는 RGB 3채널로 구성되어 있으며, 이에 맞게 필터도 다채널로 구성된다.

- 필터의 채널 수는 입력의 채널 수와 반드시 일치해야 한다
- 여러 개의 필터를 사용하면 여러 개의 특징 맵을 동시에 생성할 수 있다
- 필터 수 = 다음 레이어의 채널 수

```
입력: (H × W × C_in) → 필터 N개 → 출력: (H' × W' × N)
```

---

## 5. 1×1 Convolution의 의미

1×1 Convolution은 공간적 특징 추출은 하지 않고, **채널 간 정보를 결합**하는 데 사용된다.

- 여러 특징 맵을 가중합(weighted sum)으로 합산한다
  - 예: `w₁F₁ + w₂F₂ + w₃F₃`
- 채널 수를 줄이거나 늘리는 **채널 압축/확장** 역할을 한다
- 채널 간 상호작용을 모델링하면서도 연산량을 효율적으로 유지한다
